# MiroFish Overview

MiroFish is a next-generation AI prediction engine powered by multi-agent technology. By extracting seed information from the real world—such as breaking news, policy drafts, or financial signals—it automatically constructs a high-fidelity parallel digital world where thousands of intelligent agents interact and evolve.

## Architecture Overview

MiroFish follows a five-stage workflow pipeline:

<Steps>
  <Step title="Graph Building">
    **Knowledge Graph Construction**
    
    - Extract seed information from uploaded materials (PDF, Markdown, TXT)
    - Inject individual and collective memory into Zep Cloud
    - Build GraphRAG-powered knowledge graphs for context-aware agent reasoning
    
    *Technologies: Zep Cloud, PyMuPDF, charset-normalizer*
  </Step>
  
  <Step title="Environment Setup">
    **Agent Persona Generation**
    
    - Extract entity relationships from knowledge graph
    - Generate unique agent personas with personalities and backgrounds
    - Configure simulation parameters using LLM-powered config generation
    
    *Technologies: OASIS Profile Generator, Simulation Config Generator*
  </Step>
  
  <Step title="Simulation Execution">
    **Dual-Platform Parallel Simulation**
    
    - Run simultaneous Twitter and Reddit simulations
    - Auto-parse prediction requirements from user input
    - Dynamically update temporal memory as agents interact
    - Support configurable rounds (default: 10, production: up to 40 recommended)
    
    *Platforms: Twitter (6 actions), Reddit (13 actions)*
  </Step>
  
  <Step title="Report Generation">
    **Intelligent Analysis & Reporting**
    
    - ReportAgent equipped with rich toolset for post-simulation analysis
    - Deep interaction with simulation environment
    - Generate comprehensive prediction reports with evidence
    
    *Configuration: Max 5 tool calls, 2 reflection rounds, temperature 0.5*
  </Step>
  
  <Step title="Deep Interaction">
    **Explore the Digital World**
    
    - Chat with any agent in the simulated world
    - Query ReportAgent for additional insights
    - Inject variables dynamically to test alternative scenarios
  </Step>
</Steps>

## Key Features & Differentiators

### 1. Swarm Intelligence Emergence

Unlike traditional AI prediction models that rely on pattern matching, MiroFish captures **collective emergence** from individual agent interactions:

- **Individual Agency**: Each agent has independent personality, memory, and decision-making logic
- **Social Dynamics**: Agents interact freely, forming opinions, alliances, and conflicts
- **Emergent Patterns**: Group behaviors emerge naturally from individual actions, not pre-programmed rules

### 2. High-Fidelity Digital Twins

MiroFish constructs parallel digital worlds that mirror reality:

- **GraphRAG Integration**: Rich knowledge graphs provide context-aware reasoning
- **Long-Term Memory**: Agents remember past interactions and evolve over time
- **Dual-Platform Simulation**: Twitter and Reddit platforms capture different social dynamics

### 3. Interactive "God's-Eye View"

Users can dynamically influence simulations:

- Inject new variables and events mid-simulation
- Query agents about their thoughts and motivations
- Test "what-if" scenarios by modifying parameters

### 4. Versatile Application Scope

**Macro Level**: Decision-makers' rehearsal laboratory
- Policy impact assessment with zero real-world risk
- Public relations strategy testing
- Crisis response planning

**Micro Level**: Creative sandbox for individuals
- Novel ending predictions (e.g., Dream of the Red Chamber lost chapters)
- "What-if" explorations for entertainment
- Scenario planning for personal decisions

## Comparison: MiroFish vs. Traditional Prediction Methods

| Aspect | Traditional AI/ML | MiroFish |
|--------|-------------------|----------|
| **Methodology** | Pattern matching on historical data | Agent-based social simulation |
| **Emergence** | No emergent behaviors | Captures collective intelligence |
| **Adaptability** | Requires retraining for new scenarios | Dynamic injection of variables |
| **Explainability** | Black-box predictions | Traceable agent interactions |
| **Interaction** | Static outputs | Deep interactive exploration |
| **Scope** | Narrow, domain-specific | Universal, handles diverse scenarios |

<Note>
**Key Insight**: MiroFish doesn't just predict what *might* happen—it simulates *how* and *why* it happens through agent interactions.
</Note>
